常见问题

下载前，你可能想知道

arkNote 的笔记会上传到服务器吗？

笔记首先保存在本地。启用 Git 同步后，远端仓库保存的是加密文件，不是可直接阅读的明文笔记。

没有网络还能使用吗？

可以。编辑、预览、搜索和整理均在本地完成；只有主动同步、恢复和检查更新需要网络。

忘记密码后能找回吗？

不能。密码不会上传，也没有绕过加密的恢复通道。请妥善保存密码，并为加密数据保留离线备份。

下载链接会指向哪个版本？

本站实时读取 arkNote 的 GitHub 最新正式发行版，并按操作系统匹配对应安装包；连接失败时会自动回退到 Releases 页面。
